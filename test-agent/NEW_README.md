# Zangoh AI Agent Supervisor Workstation

This repository contains the updated Supervisor Workstation with newly implemented features including:
- Response Templates Management System
- Template Usage & Variable Injection
- SSE (Server-Sent Events) for Dashboard Metrics
- Mobile Responsive Dashboard
- Voice Input via Web Speech API

## Getting Started

To run the application:

### Backend
```bash
cd backend-starter
npm install
npm start
```
*Note: Ensure `mongodb-memory-server` is installed to run without a local Mongo DB.*

### Frontend
```bash
cd frontend-starter
npm install
npm start
```

### Testing
```bash
cd testing
npm install
node test-runner.js
```
